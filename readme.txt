== Description ==
 
This is a plugin for create and publish catalogs; it's possible select a type of catalog for create a new catalog. It's possible add registers to this catalogs and publish the catalogs into a post.

 
== Installation ==
 
1. Upload `catalog` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
 
== Frequently Asked Questions ==
 
= It's possible create several catalogs? =
 
Yes, you can create several catalogs with different name.
 
= It's possible create catalog with different structure? =
 
Yes, it's possible create catalogs from a type that define the structura, but for the initial versions only one type is available.
